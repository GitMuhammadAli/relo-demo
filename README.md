## The problem
You save brilliant tutorials and never finish them. By tomorrow, they're gone.

## The solution
relo turns a link into a structured, spaced-repetition study plan — even from videos with no captions.

## ✨ What makes it different
| | |
|---|---|
| 🎧 **Multimodal extraction** | Gemini vision reads slides; **Groq Whisper** transcribes audio — silent reels still become notes. |
| 🗓️ **Spaced repetition** | Builds a Day 1 / 3 / 7 / 14 review schedule with reminders so it sticks. |
| 🔁 **Resilient & free** | Cascading provider router (Groq → Gemini → Ollama) with a **circuit breaker**, multi-key rotation, and a SHA-256 cache cutting LLM cost ~60%. |

## 🏗️ Tech
Next.js 16 · Supabase · Groq / Gemini / Ollama · Groq Whisper · Puppeteer (Render) · token budgets.
